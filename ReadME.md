# fCStar NFT
Solidity contracts for VenusNFT Marketplace.

1. Clone repo

```shell
https://github.com/fashion-cstar/fashioncstar_nft/
```

2. Install yarn packages

```shell
cd venus-nft
yarn install
```

3. Setup and run Ganache CLI on local
```shell
npm install -g ganache-cli
ganache-cli
```

4. Run the test cases
```shell
yarn test:local
```
